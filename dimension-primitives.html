Dimension Primitives

Raw dimension scale — the foundational sizing tier. Covers rem-based and px-based steps used across spacing, sizing, and layout tokens. Values are to be defined per project.

Naming Convention Reference

Token pattern: dimension.{type}.{step}
Dimension primitives are the lowest sizing tier. Component and layout tokens reference these (e.g. spacing.input.padding.desktop → {dimension.rem-size.100}). Values are intentionally undefined and should be set per project or baseline grid (e.g. 4px, 8px).
rem-size — relative units, scales with root font size. Steps 0 → 400 (×25) + 500, 600:
0 025050075 100125150 175200225 250275300 325350375 400 500 600
px-size — fixed pixel units, device-independent. Steps 0 → 300 (×25):
0 025050075 100125150 175200225 250275300
