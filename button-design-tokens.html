Naming Convention Reference

Token pattern: button.{element}.{prominence}[.{size}][.{state}]
Component tokens sit above primitives and below compositions. Default size and default state are omitted — button.background.primary is the default-size, default-state background. Typography elements (fontSize, fontWeight) vary by size but not by state. Color elements (background, text, border, icon) vary by state but share values across sizes.
