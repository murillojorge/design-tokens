Naming Convention Reference

Token pattern: color.{hue}.{shade}
Primitives are the lowest tier — raw named color values with no semantic meaning. Semantic and component tokens reference primitives (e.g. color.input.border.error → {color.red.500}). Values are intentionally left undefined here and should be set per project or brand guidelines.
Grey scale — 0 (lightest) to 1000 (darkest):
0100200 300400500 600700800 9001000
Chromatic scales — 100 (lightest tint) to 900 (deepest shade):
100200300 400500600 700800900
Transparent — three opacity levels for overlays and scrims: light  ·  regular  ·  dark
