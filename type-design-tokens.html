Naming Convention Reference

Token pattern: type.{element}.{prominence}
Primitive tokens are the lowest tier. Each maps to one CSS property value. Semantic tokens (e.g. heading, body, caption) compose these primitives — for example, an H1 style might reference type.family.default + type.size.40 + type.weight.700 + type.lineHeight.120.
